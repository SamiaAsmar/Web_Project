# Student Collaboration Platform

## Description
The **Student Collaboration Platform** is a web application designed to simplify and organize student interactions, eliminating the need for Facebook groups. This platform allows students to share slides, tools, and course-related questions in a more structured and accessible manner. The goal is to make the academic collaboration process easier by offering a more efficient and user-friendly alternative.

## Features
- **Post Sharing**: Students can upload slides, tools, and materials related to specific courses. Others can easily find and download the resources.
- **Question Submission**: Students can submit questions about their courses and choose the type of question (e.g., multiple choice, text, checkbox, file).
- **Course Help**: A dedicated section where students can help each other with difficult course-related problems.
- **Course Tutorials**: A page for sharing tutorials and guides to help students understand complex course content.
- **User-friendly Interface**: Designed for ease of use, the platform ensures that students can navigate and interact with the content effortlessly.
